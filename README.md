# Anime Library Manager

**Anime Library Manager** — це вебзастосунок для керування власною колекцією аніме та манги.

## Основні можливості
- додавання тайтлів вручну
- пошук аніме та манги через **Jikan API**
- імпорт знайдених тайтлів у локальну базу даних
- редагування та видалення тайтлів
- створення і керування категоріями
- додавання відгуків до тайтлів
- завантаження власних зображень або використання картинок з API

## Використані технології
- **Java**
- **Spring Boot**
- **Spring Data JPA**
- **Thymeleaf**
- **PostgreSQL**
- **Maven**

## Призначення проєкту
Проєкт створений як лабораторна робота та демонструє:
- роботу з CRUD-операціями
- інтеграцію із зовнішнім API
- взаємодію з реляційною базою даних
- реалізацію серверного вебінтерфейсу

## Запуск проєкту
1. Створити базу даних PostgreSQL.
2. Вказати параметри підключення у `application.properties`.
3. Запустити проєкт через Maven або IntelliJ IDEA.
4. Відкрити у браузері:
   - `http://localhost:8080`
